Technical Specifications

Size

  • 37” vehicle height
  • 35” vehicle length
  • 24” vehicle width
  • 22” seat height

Construction

  • Aluminum sheetmetal chassis
  • Powder-paint finish

Wheels and Tires

  • 20” front wheels, quick-release, pneumatic
  • 12” rear wheel, quick-release, pneumatic
  • Anti-tip wheels

Seat

  • 20” seat base width
  • 20” seat base depth
  • 16” seat back width, tapered
  • 16” seat back height
  • High density foam core upholstered cushions
  • Seat back is adjustable and folds down for when loading

Handling

  • Bicycle style handle bar
  • Turning radius of 24"
  • Free wheel mode

Braking

  • Rear wheel internal hub brake
  • Anti roll-back

Propulsion

  • Front-and-center push-pull lever
  • Telescopic lever, up to 24” length
  • Push-rims

Powertrain

  • Enclosed double-chain drive
  • Front differential

Gearing

  • Fixed gearing, 2.5:1 ratio

Included Accessories

  • Luggage compartment

Accessory Options

  • Seat belt or harness

Compliance

  • Engineered to be ISO, ANSI/RESNA compliant
  • 250 lbs rider capacity

Weight

  • ~35 lbs, complete assembly
  • ~20 lbs, chassis, heaviest sub-assembly when loading

What's included

  • Manual
  • Seat cushions
  • Wheels
  • Integrated seat, frame, and powertrain

All specifications are subject to change without notice.
